# 24-HOUR-DIGITAL-CLOCK-USING-8051-MICROCONTROLLER-RTC-DS1307-I-C-
This report presents the design and implementation of a 24-hour digital clock using the AT89C51 (8051-architecture) microcontroller and the DS1307 Real-Time Clock (RTC) IC. DS1307 communicates with the microcontroller via the I²C (Inter-Integrated Circuit) serial protocol at a standard mode clock frequency of 100 kHz. The DS1307 is configured to produce a 1 Hz square wave output on its SQW pin, which is wired to the external interrupt line (INT1) of the 8051. This interrupt-driven architecture ensures the microcontroller reads and refreshes the display precisely once per second without requiring software managed timing loops.
The time information (hours and minutes) is retrieved from the DS1307 over I²C, stored in internal 8051 registers, and then driven to a four-digit seven-segment display (Groove 4-Digit module) through a 7447 BCD-to-seven-segment decoder operating in a multiplexed configuration. The entire firmware is written in 8051 Assembly Language and was developed in Keil µVision 5. Circuit simulation and timing verification were performed in Proteus Simulation Software using the AT89C51 device model. A Time Set Mode block is implemented using two switches, allowing manual RTC time adjustment without requiring a programming interface.
